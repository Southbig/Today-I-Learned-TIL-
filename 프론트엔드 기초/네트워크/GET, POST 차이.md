# GET

- url에 변수를 포함시켜 요청한다
- 데이터를 header에 포함하여 전송
- url에 데이터가 노출되어 보안에 취약하다
- 캐싱할 수 있다

# POST

- ur에 변수를 노출하지 않고 요청한다
- 데이터를 body에 포함시킨다
- url에 데이터가 노출되지 않아서 기본 보완은 되어있다
- 캐싱할 수 있다
